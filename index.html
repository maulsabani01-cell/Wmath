üìà Median (Nilai Tengah) - Batik Semarang

üé® Konteks Budaya: Batik Semarang memiliki motif khas seperti Tugu Muda dan Asem Arang. Mari kita cari nilai tengah dari harga batik di Pasar Johar!

Cara Mencari Median:

1. Urutkan data dari terkecil ke terbesar
2. Jika n ganjil: Median = data ke-(n+1)/2
3. Jika n genap: Median = (data ke-n/2 + data ke-(n/2+1))/2

Pengertian Modus:

Modus = Nilai yang paling sering muncul dalam data
Jika tidak ada yang berulang = Tidak ada modus
Jika ada beberapa nilai dengan frekuensi sama = Multimodus
